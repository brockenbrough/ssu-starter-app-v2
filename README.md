# SSU Starter App V2

A student-ready Next.js starter with:
- Tailwind CSS
- Supabase auth integration
- Login and register API routes
- Vitest backend tests

## Quick start

1. Copy `.env.example` to `.env.local` and fill in your Supabase values.
2. Install dependencies:
   npm install
3. Start the app:
   npm run dev
4. Run tests:
   npm test
